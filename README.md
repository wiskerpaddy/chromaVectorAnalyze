#  Saxophone Sound Analyzer (サックス音響分析)

##  概要
サックスの試奏において、2つの楽器の音響特性（芯の太さ、音の明るさ、レスポンス）を客観的な数値で比較・分析するためのWebアプリケーションです。

##  主な機能
- **比較モード (Comparison Mode)**: 2つの異なるサックス（ヴィンテージ vs モダンなど）の録音データを解析し、どちらがより「濃厚」か、あるいは「明瞭」かを判定します。
- **音量正規化ロジック**: 吹く強さやマイクとの距離による誤差を排除し、純粋な「音の成分比率」のみで楽器のポテンシャルを評価します。
- **ダークモードUI**: 演奏現場や試奏室でも目が疲れにくく、視認性の高いデザインを採用。
- **自動補正**: 発音タイミングのわずかなズレを補正し、純粋な響きの違いを抽出します。

##  技術スタック
- **Frontend**: HTML5, CSS3 (CSS Variables, Flexbox)
- **Logic**: JavaScript (ES6+)
- **Audio Engine**: Tone.js
- **Sound Synthesis**: Sampler (Piano), Reverb, Part Sequencing

##  インストール・使用方法
## 1. `index.html` をブラウザで開きます。

## 使い方
- **録音開始**: 「楽器A」を選択し、ロングトーンを録音します。
- **比較録音**: 「楽器B」を選択し、同じ条件で録音します。
- **解析**: 解析ボタンを押すと、AIが周波数成分を抽出し、比較レポートを生成します。
- **リセット**: データを消去して次の楽器のテストに移れます。

##  解析
ロジックの詳細本アプリは、以下の3つの指標で楽器を評価しています。指標- **解析内容響きの太さ (Core)**:低〜中音域のエネルギー密度を測定。音が痩せていないか判定します。
- **音の明るさ (Bright)**:高音域の倍音成分を抽出。エッジの効いた現代的な響きかを判定します。
- **レスポンス効率**:アタックの立ち上がりスピードと、入力に対する出力効率を計算します。
